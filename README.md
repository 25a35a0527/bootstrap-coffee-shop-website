# Bootstrap Coffee Shop Website

A modern and responsive coffee shop website built using Bootstrap 5 with elegant UI design, stylish sections, responsive layouts, and premium café aesthetics.

## 🚀 Features

- Fully Responsive Design
- Bootstrap 5 Layout
- Modern Navigation Bar
- Hero Section with Background Image
- Stylish Coffee Menu Cards
- About Section
- Feature Section
- Footer with Social Icons
- Smooth Hover Effects
- Mobile Friendly Interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons
- Google Fonts

## 📱 Responsive Design

This project is optimized for:
- Desktop Devices
- Tablets
- Mobile Phones

## ☕ Website Sections

- Navbar
- Hero Section
- About Us
- Coffee Menu
- Feature Section
- Footer

## 🎯 Project Objective

The objective of this project is to develop a modern responsive coffee shop website using Bootstrap 5 and frontend web development techniques.
